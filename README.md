# JavaScript Alarm Clock
 
###### By Jeff Seymour  

### Description
##### This is a simple alarm clock built with JavaScript. 


Note: Weather may not work on gh-pages properly, as API key is stored in a local .env file.

#### Technologies Used

HTML  
CSS  
JavaScript  
jQuery  
Node.js  
Gulp  
Browserify
Bower  
Moment.js

### Installation
Clone repository  
```npm install```  
```bower build```  
```gulp build```  
```gulp serve```  
View in browser at localhost:3000
